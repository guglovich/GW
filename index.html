Обновление сервера x1000
Уважаемые игроки!

В скором времени на сервер Live х1000 будут установлены новые обновления хроник.

Обновление сервера запланировано на этот вторник (10.11.2020).
Приблизительное время обновления с 6.10 по 7.00 (MSK).

Обновить клиент можно после обновления сервера х1000 через апдейтер, так же можно обновить заранее ( ссылка будет позже ), и после обновления скопировать файлы предоставленные ниже в корневую папку клиента и проверить апдейтером.

В обновление входит :
  • Система коллекций.
  • Новые умения.
Так же список некоторых изменений:
  • Исправлено умение Базовое умение Коровы ( вылетает от получения урона ).
  • Исправлена выдача бонусов от рун после смены дуала.
  • Добавлены Медовое Темное Пиво/Особый Коктейль Императора в шоп за 2 шилки/шт ( за адену -> расходники ).
  • Добавлена Серьга Линдвиора в шоп за 250 шил ( за фарм -> RB Бижа ).
  • Исправлено свечение от заточки с костюмами ассасинов без капюшона.
  • Исправлено свечение временного Рубин/Сапфира на 7 дней.
Полный список будет после установки обновления.
